# Michelin+

Web app — Hackathon **ESGI × Réseau Skolae × Michelin LB 2 Wheels** (juin 2026).

**Michelin+** : un programme communautaire & de fidélité pour cyclistes premium qui crée la demande
sans boutique directe — activation post-achat par carte+code, points & paliers, récompenses &
éditions limitées, ambassadeurs & clans.

> Repo de l'**application** (public — livrable du hackathon). Le code sera généré via **Claude design**
> puis itéré. La documentation produit/design est fournie séparément à l'équipe.

## Stack cible
Next.js 16 (App Router) · TypeScript · Tailwind · Framer Motion · Postgres (Prisma) · Docker · CI/CD (GitHub Actions) · déploiement Vercel.
