## En bref
**Michelin+** : web app premium pour cyclistes passionnés. Activation post-achat (carte+code), points & paliers, récompenses & éditions limitées, ambassadeurs & clans. **Objectif : effet WOW, design-first.**

**Style** : thème **sombre premium** unique — near-black `#08090F` + bleu Michelin `#00205B` + accent **jaune `#FCE500`** (énergie) & **or `#E8C24A`** (prestige). **Tendance : Dark Spatial Bento** (bento grid + glass 2.0 + aurora glow + compteurs kinétiques).

> ⚠️ Récupérer le **logo Michelin SVG + Bibendum** dans `public/brand/` avant le build (marque publique). Ne PAS committer les sources de charte confidentielles.
